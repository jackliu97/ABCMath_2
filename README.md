# ABCMath_2
ABCMath v2.0

### A simple CMS for

* adding/removing students
* storing classes
* building lessons, assignments and grading them
* taking attendences
* adding attachments
* question bank (beta)
* and a few other stuff that I can't think of off the top of my head right now...

### Technologies used
####PHP
* CodeIgniter 2.1
* Doctrine 2.3.5
* Symfony 2 Components (class-loader, console, dom-crawler)
* Twig

####Javascript
* Bootstrap 3
* jQuery 1.10.2
* DataTable 1.10.0
